# ReactAnyRender

## 安装说明

`yarn install react-any-render`

需要注意依赖库的安装和最低版本要求

### 依赖库

```json
{
  "@apollo/client": ">=3.0.0",
  "@fruits-chain/react-bailu": "^0.1.0",
  "antd": ">=4.0.0",
  "lodash": ">=4.0.0",
  "react": ">=17.0.2",
  "react-dom": ">=17.0.2"
}
```
